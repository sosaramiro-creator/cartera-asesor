# ⚡ PASOS RÁPIDOS: GITHUB + APP EN EL CELULAR (5 minutos)

## OPCIÓN 1: Usar solo en tu compu (sin internet)

```
1. Descargá estos 4 archivos:
   - app.html
   - index.html
   - manifest.json
   - sw.js

2. Ponelos en una carpeta (ej: "cartera")

3. Abrí app.html en el navegador

4. ✅ Listo. La app funciona offline.
```

---

## OPCIÓN 2: Publicar en GitHub (acceder desde cualquier lugar)

### PASO 1: Preparar GitHub (1 minuto)

```
1. Ve a https://github.com (crea cuenta si no la tenés)

2. Tocá "+" → "New repository" (arriba a la derecha)

3. Nombre: cartera-asesor
   Descripción: Tracker de inversiones + Asesor financiero
   ☑️ Public (para que sea accesible)
   ☑️ Add a README file

4. "Create repository" ✅
```

### PASO 2: Subir archivos (2 minutos)

```
1. En tu repositorio, tocá "Add file" → "Upload files"

2. Arrastrá o seleccioná:
   - app.html
   - index.html
   - manifest.json
   - sw.js

3. Mensaje: "Agregar app inicial"

4. "Commit changes" ✅
```

### PASO 3: Activar GitHub Pages (1 minuto)

```
1. En tu repositorio: Settings → Pages (menú izquierdo)

2. Source: "Deploy from a branch"
   Branch: main
   Folder: / (root)

3. "Save" ✅

4. Espera 1-2 minutos...

5. Verás el mensaje: "Your site is published at:"
   → https://TU_USUARIO.github.io/cartera-asesor

¡LISTO! 🎉
```

### PASO 4: Instalar como app en el celular (1 minuto)

#### 📱 iPhone (iOS)
```
1. Abrí en Safari:
   https://TU_USUARIO.github.io/cartera-asesor

2. Compartir (↗️) → "Agregar a Pantalla de inicio"

3. ✅ App en tu home
```

#### 📱 Android (Chrome)
```
1. Abrí en Chrome:
   https://TU_USUARIO.github.io/cartera-asesor

2. Menú (⋮) → "Instalar app"

3. ✅ App en tu home
```

## SI ALGO NO FUNCIONA

**"No se ve la app"** → Esperá 2 minutos, recargá página (Ctrl+F5)

**"No se instala como app"** → Necesita HTTPS. GitHub Pages la proporciona automáticamente.

**"Perdí mis datos"** → Los datos se guardan en localStorage de tu navegador. No se pierden.
